# Notch

A Dynamic Island-inspired overlay for Android that brings media controls and live interactions directly to your camera cutout.

Built as a lightweight overlay service with a focus on smooth animations, privacy, and customization.

## Features

- Dynamic Island-style overlay
- Music controls with album artwork, title, artist, playback controls, and progress tracking
- Tap to expand
- Double-tap to switch modules
- Long-press for settings
- Swipe down to dismiss
- Automatic collapse after inactivity
- Boot auto-start
- Notification and media session integration
- Notch calibration for different devices
